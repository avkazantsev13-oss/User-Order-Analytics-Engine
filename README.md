# User-Order-Analytics-Engine
Лёгкий, но мощный модуль для моделирования связей между пользователями и их заказами с мгновенным получением агрегированной статистики. Идеально подходит для малого бизнеса, стартапов и разработчиков, которым нужен надёжный инструмент для отслеживания суммы заказов по каждому клиенту.
